**Typora常见使用防范**

- 标题

  ```
  #       一级标题    
  ##      二级标题    
  ###     三级标题    
  ####    四级标题    
  #####   五级标题    
  ######  六级标题
  ```

- 引用

  ```
  > 引用内容1
  >> 引用内容2
  >>> 引用内容3
  ```

  - 效果

  > 引用1
  >
  > > 引用2
  > >
  > > > 引用3

- 单行代码

  `string s = "zhaokun"`

- 代码块

  ```java
  string s = "zhaokun"
  string s = "zhaokun"
  ```
